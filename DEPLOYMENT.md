# Deployment Instructions

## Deploying to Vercel

1. Go to the [Vercel website](https://vercel.com).  
2. Click on the "Import Project" button.  
3. Choose your GitHub repository (Lalinterna/primeros-pasos).
4. Configure any necessary settings for the deployment.  
5. Click on the "Deploy" button to complete the deployment.

## Deploying to Railway

1. Visit the [Railway website](https://railway.app).
2. Click on the "New Project" button.  
3. Select the option to deploy from GitHub and choose your repository (Lalinterna/primeros-pasos).
4. Set up any environment variables and configuration needed for your project.
5. Click the "Deploy" button to start the deployment process.

For any further customizations or settings, refer to the official documentation of Vercel and Railway.
